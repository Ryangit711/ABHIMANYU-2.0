## PLATFORM: Greenhouse

### How It Works
- **Scorecard-based:** Each interviewer submits a scorecard (1-4 scale) per competency
- **Weighting:** The hiring manager can weight competencies (e.g., "Leadership" = 2x, "Technical" = 1x)
- **Source tagging:** Application source tracked (referral, LinkedIn, company board, recruiter)
- **Rejection cooldown:** 6 months default cooldown for same role, no cooldown for different role
- **Talent pool:** Auto-enrolled unless you opt out. If you decline an offer, you enter talent pool with notes

### Insider Tips
- **Referral matters** — referral applications get a source tag that gives hiring manager a quick "trust" filter
- **Culture-add questions** — many companies use custom application questions. These are scored separately and can auto-fail before anyone reads your resume
- **Multi-app handling:** Greenhouse shows ALL your applications to the recruiter simultaneously. If they're different titles at different levels, it looks like initiative. Same title = spray-and-pray flag
- **Withdraw gracefully:** Use the withdraw button instead of ghosting. Withdrawn apps don't count toward cooldown

### How To Score High
- Use exact JD keywords in your resume — Greenhouse's keyword parser weights exact matches higher than synonyms
- Answer custom questions with company-specific language (mirror their JD verbs)
- Apply for no more than 3 roles at the same company, spaced 1+ week apart

## PLATFORM: Workday

### How It Works
- **Global profile:** One Workday profile follows you across companies that use Workday (if they share a tenant, which most don't — but multi-nationals do)
- **Talent pool auto-enrollment:** When you apply, you're automatically enrolled in the talent pool unless you deselect (un-check the box)
- **Rejection handling:** If rejected, you stay in talent pool. Recruiters can search talent pool for future roles
- **Multi-app handling:** Workday treats each application as independent. BUT recruiters can see your full application history with the company
- **Assessment integration:** Some companies embed assessments (PwC's digital fitness test, Deloitte's business chemistry) inside Workday

### Insider Tips
- **Un-check talent pool enrollment** unless you want recruiter spam (or you DO want it — useful for passive reach)
- **Don't apply to more than 2 roles simultaneously** — Workday shows them chronologically and recruiters can see you applied to everything
- **Workday parses PDFs well** — but keep it simple (no tables, no columns, no graphics)
- **Workday recommends** — the "Recommended for you" algorithm is based on title match + location + skills. Optimize your profile for the algorithm

### Canadian Specifics
- Canadian Workday tenants are often separate from US tenants (privacy law)
- If rejected for a Vancouver role, you won't automatically appear in the US recruiter's talent pool

## PLATFORM: SAP SuccessFactors

### How It Works
- **Business unit silos:** Each business unit (Strategy, Audit, Tax, Advisory) has a separate recruiting pipeline
- **Multi-app handling:** Applying to Strategy AND Audit is fine (different pipelines). Applying to two Strategy roles = visible to same recruiter
- **Talent pool:** RFx (Recruitment Function) pools candidates by competency
- **Scoring:** Uses a weighted scoring algorithm based on job requirements mapping

### Insider Tips
- **Target the right business unit** — Deloitte Strategy & Operations is different from Deloitte Consulting, which is different from Deloitte Digital. Apply to the correct one
- **Internal mobility** — SuccessFactors is designed for internal mobility. External candidates look like external. Don't try to bypass
- **Rejection silence** — If your application status changes to "No Longer Under Consideration," wait 6 months before reapplying to the same BU
- **Talent community** — Join Deloitte's talent community separately from applying. Talent community members sometimes get faster screening

## PLATFORM: Lever

### How It Works
- **Source tracking is prominent** — recruiters see exactly where you came from (referral, LinkedIn ad, cold apply)
- **Kanban-style pipeline** — candidates move through stages visually. Recruiters use this daily
- **Notes visible** — internal notes about you are visible to all interviewers
- **Rejection options** — Lever has a "passive" rejection (talent pool) vs "active" rejection (do not contact)

### Insider Tips
- **Referral source is the strongest signal** — Lever's UI puts source front and center
- **Don't cold-apply if you can avoid it** — Lever makes cold applications obvious. Find a referral or recruiter contact first
- **Resume parsing** — Lever uses basic keyword matching. Simple formatting wins

## PLATFORM: UltiPro (MNP)

### How It Works
- **Simpler system** — less ATS gaming needed
- **Manual review** — smaller companies using UltiPro often manually review every application
- **Keyword matching is basic** — focus on resume quality over keyword density
- **No complex scoring** — human reads your resume, not an algorithm

### Insider Tips
- **Cover letter matters more** — at UltiPro shops, humans read cover letters. Write a good one
- **Follow up directly** — smaller firm, find the hiring manager on LinkedIn and follow up after applying

## PLATFORM: Ashby

### How It Works
- **Modern API-driven ATS** — Ashby is built for growth-stage companies (1Password, Procurify, Trulioo)
- **Culture-add questions** — companies configure custom application questions scored separately
- **Source tracking** — visible to recruiters, referral source is strongest signal
- **Fast turnaround** — startups using Ashby typically respond within 3-7 days

### Insider Tips
- **Custom questions matter** — Ashby allows unlimited custom questions. Answer each one with company-specific language
- **Resume parsing** — good at extracting structured data. Simple formatting preferred
- **Multi-app handling** — shows all your apps on one dashboard. Coherent titles = initiative. Same title twice = spray-and-pray
- **No cooldown between different roles** — but wait 1 week minimum

## PLATFORM: SmartRecruiters

### How It Works
- **Task-based pipeline** — recruiters move candidates through stages with checklist-style UI
- **AI matching** — SmartRecruiters uses AI to score candidates against JD keywords
- **Talent pool** — auto-enrolls candidates, recruiters search pool for future roles
- **Source tracking** — visible to recruiters

### Insider Tips
- **Keyword density matters** — SmartRecruiters' AI scores resumes against JD. Use exact JD phrases
- **Easy to get lost** — large companies (AECOM) get high volume. Apply early in posting window
- **Talent community** — join before applying for passive reach from recruiters

## PLATFORM: BambooHR

### How It Works
- **Simple HRIS with recruiting module** — used by small/mid Canadian companies
- **Human review** — less automated filtering, humans read applications
- **Basic keyword matching** — no complex AI scoring
- **Email notifications** — recruiters get email for each application

### Insider Tips
- **Cover letter helps** — humans read them at BambooHR shops
- **Follow up directly** — smaller companies, find hiring manager on LinkedIn
- **Simple resume wins** — standard DOCX, no complex formatting needed

## PLATFORM: Breezy

### How It Works
- **Visual pipeline** — drag-and-drop kanban board, recruiter-driven
- **Source tracking** — shows where candidates came from
- **Basic ATS** — used by early-stage startups
- **Fast cycle** — decisions made quickly (1-2 weeks)

### Insider Tips
- **Direct apply > aggregator** — Breezy shows source prominently
- **Referral is king** — referrals tagged differently from cold apps
- **Simple resume** — standard format, no tables

## PLATFORM: Personio

### How It Works
- **EU-headquartered** — used by European companies with Canadian offices
- **All-in-one HR** — recruiting + HRIS in one platform
- **Compliance-focused** — GDPR-ready, privacy-first design
- **Scorecard-based** — interviewers submit structured scores

### Insider Tips
- **European privacy norms** — don't expect fast response (EU work culture)
- **Resume format** — standard DOCX, Calibri, 0.75in margins
- **Multi-app** — different roles at same company = fine, different departments = invisible

### Insider Tips

- **No special insider knowledge needed** — treat as generic modern ATS

## PLATFORM: Avature

### How It Works
- **Enterprise CRM-ATS hybrid** — used by IBM, large enterprises
- **Relationship-based** — Avature tracks candidate relationships over time
- **Talent pool is primary** — recruiters search talent pool before posting roles
- **Complex scoring** — multi-dimensional candidate scoring

### Insider Tips
- **Join talent community first** — Avature shops love this. Makes you an insider before applying
- **Relationship matters** — connect with recruiters on LinkedIn before applying
- **Enterprise slow cycle** — expect 4-8 weeks from apply to offer

## PLATFORM: Pinpoint

- **Collaborative scoring** — team input on each candidate
- **Fast feedback** — designed for growth-stage velocity

### Insider Tips
- **Values alignment** — study company values page, weave into application
- **Referral helps** — Pinpoint shows source prominently
- **Standard DOCX** — clean format, no special rules

## PLATFORM: JibeApply (Oracle)

### How It Works
- **Oracle product** — enterprise-focused, legacy integration
- **Multi-stage pipelines** — structured hiring process
- **Auto-rejection** — keyword threshold auto-rejects weak matches
- **Taleo alternative** — similar to Taleo but cloud-based

### Insider Tips
- **Keyword optimization critical** — auto-rejection if under threshold. Use exact JD language
- **Test before submit** — verify DOCX renders correctly (Jibe can mangle formatting)
- **Multi-app visible** — same recruiter sees all apps

## PLATFORM: Comeet

### How It Works
- **Mid-market ATS** — used by growing companies
- **Structured interviews** — predefined scorecards per role
- **Collaborative** — team input on hiring decisions
- **Calendar-driven** — scheduling integrated into platform

### Insider Tips
- **Standard DOCX works** — no special formatting needed
- **Process transparency** — candidates can see their stage (good for follow-up timing)
- **1-2 apps max** — Comeet shows history prominently

## PLATFORM: Oracle Cloud (Oracle HCM)

### How It Works
- **Enterprise cloud HCM** — used by large organizations
- **Global talent profile** — one profile across all Oracle Cloud companies (if same tenant)
- **AI-driven matching** — Oracle AI scores candidates against requirements
- **Talent pool auto-enrollment** — default behavior

### Insider Tips
- **Multi-tenant nuance** — Canadian vs US tenants are separate (privacy law)
- **Keyword density** — Oracle AI scores keyword match rates
- **Talent community** — join before applying for passive algorithm recommendation
- **Multi-app strategy** — 2-3 related roles, 1+ week spacing
